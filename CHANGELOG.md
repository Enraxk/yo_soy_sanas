## 👥 Contribuciones

### **Estructura de Commits**
Seguimos la convención de **Conventional Commits** con emojis:

```bash
✨ feat(scope): descripción de nueva característica
🐛 fix(scope): descripción de corrección
📝 docs(scope): cambios en documentación
♻️ refactor(scope): refactorización de código
🎨 style(scope): cambios de formato/estilo
✅ test(scope): añadir o corregir tests
📦 chore(scope): cambios en build o dependencias
```

### **Scopes Utilizados**
- `chakras` - Sistema de chakras y datos espirituales
- `carousel` - Carrusel y navegación
- `components` - Componentes React
- `auth` - Sistema de autenticación
- `ui` - Interfaz de usuario y estilos
- `config` - Configuración y setup
- `docs` - Documentación y README
